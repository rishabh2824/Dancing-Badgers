# Dancing Badgers 3.0

## Overview
Dancing Badgers is a Java-based graphic application developed using the Processing library. This project involves creating a dynamic GUI with interactive elements such as dancing badgers and clickable basketballs. The application demonstrates object-oriented programming concepts, including inheritance, interfaces, and polymorphism.

## Features
- Interactive GUI with animated badgers and basketballs.
- Utilizes Processing library for graphics rendering.
- Implements object-oriented design principles.
- Designed for ease of maintenance and extensibility.

### Setup
1. Create a new java project and add the core.jar file to your project build path.
2. Extract the images folder and other java files and add to your project.
3. Run the Dancing Badgers.java file and see the magic!
